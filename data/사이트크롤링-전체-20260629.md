- [ ] 과정별 수강료/기간/정원 정보 부재 → 추가 필요
- [ ] 수강생 후기/졸업작품 섹션 없음
- [ ] 온라인 수강신청 폼 없음 (전화만 가능)
- [ ] 모바일 UX 개선 필요 (WordPress 기반)
- [ ] 원장/부원장 프로필 사진 없음
- [ ] 갤러리 카테고리 분류 미흡
