이론 — Midjourney vs Firefly(00:00 – 12:00)
지난 주에는 Midjourney로 이미지를 생성했습니다. 이번 주는 Adobe Firefly와 Photoshop AI입니다. '왜 또 다른 도구를 배우나'라고 생각하실 수 있는데, 두 도구는 쓰임새가 완전히 다릅니다.

Midjourney — 처음부터 이미지를 '창작'하는 도구
  · 새로운 비주얼을 만들 때 (무드보드, 컨셉 시뮬레이션)
  · 예술적·광고적 느낌이 강함
  · 실제 사진 편집 불가

Adobe Firefly / Photoshop AI — 실제 사진을 '편집'하는 도구
  · 이미 찍은 사진에 요소를 추가/제거할 때
  · 배경 교체, 오브젝트 삽입, 캔버스 확장
  · 실사 품질 유지 + Adobe 저작권 안전

정리하면 Midjourney는 '없는 것을 만들 때', Firefly/Photoshop AI는 '있는 사진을 바꿀 때' 씁니다. 실무에서는 두 가지를 함께 씁니다. Midjourney로 배경을 만들고, Photoshop AI로 실제 제품 사진과 합성하는 식으로요.
Adobe Firefly는 Adobe Stock 이미지로만 학습되어 저작권이 안전합니다. 상업적 결과물에 써도 법적으로 문제없는 유일한 주요 AI 이미지 도구입니다.
실습 ① — Firefly 웹 텍스트→이미지(12:00 – 28:00)
  1. firefly.adobe.com 접속 → Adobe 계정으로 로그인
  2. '텍스트를 이미지로' 클릭
  3. 아래 프롬프트 입력 후 생성

[Firefly 프롬프트 예시]

A luxury skincare serum bottle on a white marble surface,
soft morning light from the left, minimalist product photography,
clean white background, high detail

오른쪽 패널 설정:
· 종횡비: 4:5 (인스타 세로)
· 콘텐츠 유형: 사진
· 스타일: 없음 (기본)
· 조명: 스튜디오 조명

Midjourney와 비슷하지만, Firefly는 Adobe의 인터페이스라 훨씬 직관적입니다. 오른쪽 패널에서 스타일, 조명, 색조를 버튼으로 쉽게 바꿀 수 있어요.
생성된 이미지 중 1장 선택 → '편집' 클릭 → Photoshop에서 열기
Firefly 무료 계정은 월 25크레딧입니다. Creative Cloud 구독자는 무제한에 가깝습니다. 오늘 수업은 Photoshop 실습이 메인이므로 Firefly에서 이미지 2~3장만 생성하고 넘어갑니다.
실습 ② — Photoshop Generative Fill 기초(28:00 – 50:00)
이제 Photoshop을 엽니다. Generative Fill은 2023년부터 추가된 AI 기능으로, 선택한 영역에 AI가 이미지를 채워주는 기능입니다.
실습용 제품 사진 Photoshop에서 열기
  1. Photoshop 열기 → 실습 제품 사진 파일 드래그 앤 드롭
  2. 왼쪽 도구에서 올가미 도구(L) 선택
  3. 제품 옆 빈 공간 또는 배경 일부 선택
  4. 하단 컨텍스트 바 'Generative Fill' 클릭
  5. 텍스트창에 추가하고 싶은 요소 입력
  6. 생성 버튼 클릭 → 3가지 변형 결과 확인

프롬프트 예시별 실습:

① 소품 추가: dried lavender flowers, elegant
② 배경 변경: white marble texture background
③ 조명 추가: soft warm sunlight from window

레이어 패널 확인:
· Generative Fill은 자동으로 새 레이어로 생성됨
· 원본 사진은 절대 손상되지 않음
· 속성 패널에서 3가지 변형 중 선택 가능

레이어 패널을 보세요. 원본 레이어는 그대로 있고, 위에 AI 생성 레이어가 새로 생겼습니다. 마음에 안 들면 삭제하면 끝입니다. 원본이 절대 손상되지 않아요. 이게 Generative Fill의 핵심 장점입니다.
Generative Fill 프롬프트는 한국어도 됩니다. '라벤더 꽃다발', '대리석 배경'처럼 입력해도 잘 됩니다. 단, 영어가 더 정확한 결과를 냅니다.
실습 ③ — Generative Expand (캔버스 확장)(50:00 – 68:00)
Generative Expand는 이미지 바깥쪽을 AI로 채우는 기능입니다. 세로로 찍은 사진을 가로 배너로 만들거나, 여백을 추가할 때 씁니다.
  1. 자르기 도구(C) 선택
  2. 캔버스 바깥쪽으로 드래그해서 영역 확장
  3. 상단 옵션바에서 'Generative Expand' 선택 확인
  4. Enter → 생성 대기
  5. 속성 패널에서 3가지 변형 중 선택

실전 활용 케이스:

· 세로(4:5) 제품 사진 → 가로(16:9) 유튜브 썸네일로 확장
· 인스타 정사각(1:1) → 배너(3:1)로 확장
· 제품 위쪽 여백 추가 → 텍스트 공간 확보
· 인물 사진 좌우 공간 추가 → 잡지 레이아웃 적합하게

SNS용으로 찍은 세로 사진을 유튜브 썸네일로 써야 할 때 정말 많죠? 이전엔 배경을 새로 찍거나 크롭해야 했는데, 이제는 30초 만에 해결됩니다.
실습 ④ — Remove Tool (배경 요소 제거)(68:00 – 82:00)
마지막으로 Remove Tool입니다. 배경에 있는 불필요한 요소를 브러시로 칠하면 AI가 지워줍니다.
  1. 왼쪽 도구에서 복구 브러시 그룹 길게 클릭 → Remove Tool 선택
  2. 제거하고 싶은 요소 위에 브러시로 칠하기
  3. 손을 떼면 자동으로 제거됨
  4. 결과가 마음에 안 들면 Ctrl+Z 후 다시 시도
실습: 배경에 불필요한 요소(전선, 그림자, 스티커 등) 제거 실습
콘텐츠 인식 채우기(Content-Aware Fill)와 비슷하지만 훨씬 정확합니다. 특히 복잡한 배경에서 작은 물체를 지울 때 강력합니다.
Remove Tool로 지운 영역이 어색하다면, 추가로 Generative Fill에 '텍스트 없이 (no text, no object)'처럼 입력해서 한 번 더 보정하면 됩니다.
정리(82:00 – 90:00)
오늘 배운 것 정리합니다. Adobe Firefly는 텍스트로 이미지를 생성합니다. Photoshop Generative Fill은 선택 영역에 AI로 요소를 추가합니다. Generative Expand는 캔버스를 자연스럽게 확장합니다. Remove Tool은 원치 않는 요소를 제거합니다. 다음 시간에는 이 기능들을 전부 합쳐서 실제 제품 사진과 Midjourney 배경을 합성하는 실전 실습을 합니다.
실습 과제
과제 — 제품 사진 3가지 AI 편집 적용
본인 소유 또는 수업에서 제공된 제품 사진으로 아래 3가지를 각각 적용하고 결과를 저장해오세요.
  • Generative Fill로 소품 또는 배경 요소 1가지 추가
  • Generative Expand로 캔버스 비율 변경 (세로→가로 또는 반대)
  • Remove Tool로 배경 잡티 제거
  • 전·후 비교 스크린샷 각 1장 (총 6장)
