Gemini 이미지 분석 & 콘텐츠 기획

Google Gemini의 이미지 분석 기능과 검색 연동으로 사진 레퍼런스를 분석하고 콘텐츠 캘린더를 자동화합니다.

수업 시간90분
대상 도구Google Gemini
준비물구글 계정, 레퍼런스 사진 3장
난이도★★☆☆☆ 기초
학습 목표
  • Gemini에 이미지를 업로드해 조명·구도·컬러 분석을 요청할 수 있다
  • 레퍼런스 사진을 분석해 촬영 가이드를 추출할 수 있다
  • Gemini의 Google 검색 연동으로 최신 트렌드를 리서치할 수 있다
  • 한 달치 SNS 콘텐츠 캘린더를 자동으로 생성할 수 있다
수업 흐름
00:00이론 — Gemini의 강점 (이미지 이해 + 구글 연동)
10:00실습 ① — 레퍼런스 사진 이미지 분석
35:00실습 ② — 트렌드 리서치 & SEO 키워드
55:00실습 ③ — 한 달 SNS 콘텐츠 캘린더 자동 생성
75:003가지 도구 정리 & 사용 루틴 설계
강의 스크립트
이론 — Gemini 강점(00:00 – 10:00)
오늘은 Google Gemini입니다. Gemini는 구글이 만든 AI로 두 가지 큰 강점이 있습니다. 첫째, 이미지를 이해하고 분석하는 능력. 둘째, 구글 검색과 연동되어 최신 정보를 가져올 수 있는 것.

Gemini 강점 요약:
① 이미지 업로드 → 조명·구도·컬러·분위기 분석
② 구글 검색 연동 → 최신 트렌드·뉴스·통계 가져오기
③ 다국어 → 해외 레퍼런스 영문 분석 후 한국어 요약
④ 구글 드라이브·문서 연동 가능 (Google One AI 구독 시)

ChatGPT와 Claude가 텍스트 중심이었다면, Gemini는 '사진을 보여주고 분석받는' 용도에 특히 강합니다.
실습 ① — 레퍼런스 이미지 분석(10:00 – 35:00)
준비한 레퍼런스 사진을 Gemini에 올려봅시다. 화면 아래쪽에 이미지 첨부 버튼이 있습니다.
  1. gemini.google.com 접속 → 구글 계정 로그인
  2. 입력창 오른쪽 이미지 아이콘 클릭 → 레퍼런스 사진 업로드
  3. 아래 프롬프트 입력

[이미지 분석 프롬프트]

이 사진을 사진작가 관점에서 분석해주세요.

분석 항목:
1. 조명 — 광원 방향, 광원 종류 추정 (자연광/인공광, 소프트/하드), 빛과 그림자 비율
2. 구도 — 촬영 앵글, 구도 방식 (3분할/중앙/리딩라인 등), 여백 활용
3. 컬러 — 주조색, 보조색, 전체 색온도 (따뜻함/차가움)
4. 분위기 키워드 — 이 사진의 느낌을 5개 단어로
5. 이 사진을 재현하기 위한 촬영 세팅 추천

이렇게 나온 분석을 인쇄해서 촬영 현장에 가져가면, 레퍼런스를 그대로 재현하는 데 큰 도움이 됩니다.
여러 레퍼런스 사진을 한 번에 올려서 '이 사진들의 공통 스타일을 분석해줘'라고 하면 무드보드의 방향을 잡는 데 유용합니다.
실습 ② — 트렌드 리서치(35:00 – 55:00)
Gemini는 구글 검색과 연결되어 있어서 최신 정보를 가져올 수 있습니다.
아래 프롬프트 입력

[트렌드 리서치 프롬프트]

2026년 뷰티·화장품 광고 사진의 최신 트렌드를 조사해주세요.

포함할 내용:
1. 지금 인스타그램·핀터레스트에서 유행하는 뷰티 사진 스타일 3가지
2. 각 스타일의 조명·컬러·소품 특징
3. 국내 브랜드 중 이 트렌드를 잘 적용한 사례
4. 이 트렌드에 맞게 촬영할 때 주의할 점

ChatGPT나 Claude는 학습 데이터 기준일이 있어서 최신 트렌드를 모를 수 있습니다. 이럴 때 Gemini의 구글 연동이 빛납니다.
실습 ③ — 콘텐츠 캘린더 자동 생성(55:00 – 75:00)
아래 프롬프트 입력

[콘텐츠 캘린더 프롬프트]

인스타그램 계정 운영을 위한 8월 한 달 콘텐츠 캘린더를 만들어주세요.

계정 정보:
- 업종: 사진 스튜디오 / 사진 강좌
- 타깃: 20~40대 사진에 관심 있는 일반인
- 포스팅 빈도: 주 3회 (월·수·금)

각 포스팅마다:
- 날짜
- 콘텐츠 주제
- 사진 방향 (어떤 사진을 올릴지)
- 캡션 방향 (어떤 메시지를 담을지)
- 추천 해시태그 5개

표 형식으로 만들어주세요.

이렇게 만든 캘린더를 노션이나 구글 스프레드시트에 옮기면 한 달 운영 계획이 완성됩니다.
1~2주차 정리 — 3가지 도구 사용 루틴(75:00 – 90:00)
지난 2주 동안 ChatGPT, Claude, Gemini 세 가지를 배웠습니다. 실무에서 이 도구들을 어떻게 쓰면 좋을지 루틴을 정리해드릴게요.

촬영 기획 시작할 때:
1. Gemini로 트렌드·레퍼런스 리서치
2. ChatGPT로 무드보드 컨셉·체크리스트 빠르게 뽑기
3. Claude로 클라이언트 제안서·시나리오 다듬기

촬영 후 콘텐츠 작업:
1. Gemini로 완성 사진 분석 (다음 촬영 개선점 찾기)
2. ChatGPT로 SNS 캡션 5버전 생성
3. Claude로 뉴스레터·블로그 글 작성

다음 시간부터는 이미지 생성 파트로 넘어갑니다. Midjourney로 실제 이미지를 만드는 거예요. 사진학원인데 왜 AI 이미지를 만드냐고요? 촬영 전 비주얼 시뮬레이션, 무드보드에 바로 활용할 수 있기 때문입니다.
실습 과제
과제 — 레퍼런스 분석 & 콘텐츠 캘린더
좋아하는 사진 계정에서 레퍼런스 3장을 고르고, Gemini로 분석한 뒤 다음 달 콘텐츠 캘린더를 만들어오세요.
  • 레퍼런스 3장 Gemini 업로드 → 공통 스타일 분석 요청
  • 분석 결과를 참고해 내 계정에 적용할 방향 정리
  • Gemini로 8월 콘텐츠 캘린더 자동 생성
  • 캘린더 중 가장 마음에 드는 포스팅 1개 → ChatGPT로 캡션 5버전 생성
