실습 ① — Midjourney 배경 생성(08:00 – 20:00)
합성할 배경부터 만듭니다. 핵심은 '제품 사진과 조명 방향을 맞추는 것'입니다. 제품 사진의 빛이 왼쪽에서 온다면, 배경도 왼쪽에서 빛이 와야 합성이 자연스럽습니다.
Midjourney에서 배경 이미지 생성

배경 프롬프트 작성 팁:

· 조명 방향 명시: soft light from the left side
· 피사체 없는 배경: empty surface, no objects, no people
· 적절한 여백: product placement space in center
· 고해상도: --q 2 --ar 4:5

예시 프롬프트:
empty white marble table, soft morning light from left,
minimalist background for product photography,
no objects --ar 4:5 --q 2 --style raw

생성 후 U 버튼으로 업스케일 → 이미지 저장. 이 이미지가 합성의 배경이 됩니다.
배경 이미지는 최대한 심플하게 만드세요. 배경이 복잡할수록 합성이 어색해집니다. 제품이 주인공이니까요.
실습 ② — 제품 피사체 분리(20:00 – 38:00)
이제 제품 사진에서 제품만 잘라냅니다. Photoshop의 Subject Selection이 AI로 자동 선택해줍니다.
  1. 제품 사진 Photoshop에서 열기
  2. 상단 메뉴 → 선택 → 피사체 (Select > Subject)
  3. 선택 영역 확인 — 점선으로 제품 외곽이 선택됨
  4. 선택 및 마스크(Select and Mask) 열기
  5. 가장자리 다듬기 브러시로 세밀한 부분(뚜껑 경계, 반사면) 보정
  6. 출력: 레이어 마스크 선택 → 확인
  7. 레이어 패널에서 마스크 아이콘 확인
유리 소재나 반투명 제품은 자동 선택이 어렵습니다. 이럴 때는 채널 선택법이나 펜 도구(P)로 수동 패스를 만드는 것이 더 정확합니다. 오늘은 기초이므로 Subject Selection으로 진행합니다.

레이어 마스크 확인 방법:
레이어 패널에서 레이어 썸네일 옆에 검정/흰색 마스크가 보이면 성공.
흰색 = 보이는 영역 (제품), 검정 = 숨겨진 영역 (배경)

배경이 사라지고 제품만 남았죠? 이 레이어를 그대로 배경 위에 올립니다.
실습 ③ — 배경 합성 & 위치 조정(38:00 – 55:00)
  1. Midjourney 배경 이미지 Photoshop에서 열기
  2. 제품 레이어를 배경 파일로 드래그 앤 드롭
  3. 레이어 패널: 제품 레이어가 배경 레이어 위에 있는지 확인
  4. Ctrl+T (Free Transform) → 크기와 위치 조정
  5. Shift 키 누른 채 드래그 → 비율 유지하며 크기 조절
  6. 위치는 배경의 밝은 부분(빛이 닿는 면) 쪽에 배치
배경 위에 제품이 올라갔습니다. 아직 어색하죠? 배경과 제품의 색감이 다르고, 그림자가 없어서 '붕 떠 있는' 느낌이 납니다. 이걸 다음 두 단계에서 해결합니다.
중간 점검: 레이어 순서 확인, 제품 크기·위치 재조정
제품이 너무 크거나 작으면 실제감이 없습니다. 배경의 다른 오브젝트(테이블 면, 그릇 등)와 원근감을 맞추는 게 핵심입니다.
실습 ④ — 그림자 & 반사광 추가(55:00 – 72:00)
합성의 완성도는 그림자에서 결정됩니다. 그림자가 없으면 무조건 합성티가 납니다.

방법 1 — 레이어 스타일 드롭 섀도 (빠른 방법)

  1. 제품 레이어 더블클릭 → 레이어 스타일 열기
  2. 드롭 섀도(Drop Shadow) 체크
  3. 각도: 배경 빛 방향에 맞게 (예: 135도)
  4. 거리: 30~50px, 크기: 40~60px, 불투명도: 40~60%
  5. 확인

방법 2 — 수동 그림자 레이어 (정교한 방법)

  1. 제품 레이어 아래에 새 레이어 추가
  2. 검정으로 제품 바닥 형태를 타원형으로 그리기
  3. 필터 → 흐림 효과 → 가우시안 흐림 (30~50px)
  4. 레이어 불투명도 40% 내외로 조절
  5. 제품 위치에 맞게 살짝 이동 (빛 반대 방향)
방법 2가 더 자연스럽습니다. 실무에서는 두 방법을 섞어 씁니다. 드롭 섀도로 윤곽 그림자를 만들고, 수동 레이어로 바닥 그림자를 추가합니다.
추가 실습: 반사광 (Specular Highlight) — 제품 측면에 하이라이트 레이어 추가
실습 ⑤ — 색감 매칭 & 내보내기(72:00 – 90:00)
마지막으로 제품과 배경의 색감을 통일합니다. 배경이 따뜻한 노란빛인데 제품이 차가운 흰빛이면 어색합니다.
  1. 제품 레이어 선택 → 상단 메뉴 이미지 → 조정 → 색조/채도
  2. 배경 분위기에 맞게 색온도 미세 조정 (색조 ±10 이내)
  3. 추가로: 레이어 → 새 조정 레이어 → 곡선(Curves)
  4. 클리핑 마스크(Ctrl+Alt+G)로 제품 레이어에만 적용

내보내기 3종:

· SNS용: 파일 → 내보내기 → 웹용으로 저장 / JPEG 80% / 1080×1350px
· 웹용: JPEG 90% / 1200px 기준
· 인쇄용: TIFF 또는 PSD 원본 / 300dpi / CMYK 변환

완성입니다. 오늘 만든 결과물, 처음에 제가 보여드린 예시와 비슷하게 나왔죠? 이 워크플로 — Midjourney 배경 생성, 피사체 분리, 합성, 그림자, 색감 매칭, 내보내기 — 이걸 5번만 반복하면 자기 것이 됩니다.
