## 3. 수강생 평가·후기 분석 (공개 후기 기반)

> 인스타 댓글은 계정 소유자만 전수 확인 가능 → 여기서는 **공개된 실제 수강 후기**를 분석했습니다.

### 3-1. 실제 수료 후기 (네이버 블로그, 입문반 수료생)

**만족 포인트 (그대로 마케팅 소재):**
- "선생님도 너무 좋았고" — 강사 만족도 높음
- "카메라 작동법과 **포토샵**을 배울 수 있어서 너무 좋았다"
- "스튜디오처럼 배경 두고 **조명** 켜 찍는 게 최고였다. **엄청 비싼 조명**을 써 볼 기회까지"
- "사진 잘 찍는다고 **칭찬**받은 것, 좋은 사람들 만난 것 다 감사"
- 원장님 인상: "구직자반은 **목숨 걸고** 하는 겁니다" (강한 카리스마)
- "**구직자반 + 내일배움카드(국비지원)** 할인" → 퇴사 후 진지하게 재수강 고민

**개선 신호 (참고):**
- "내가 찍은 사진을 **포폴로 쓰고 싶었지만** 모델 동의가 부담스러워 못 올림" → 수강생 작품 활용/동의 프로세스 정비 필요
- 온라인 리뷰 플랫폼(프롬피·강남엄마)에 **후기 0개** → 만족도는 높은데 **공개 후기로 축적이 안 됨**

### 3-2. 학원 1차 후기 자료 (수료생 인터뷰 대본 기반)

> 학원이 준비한 후기 영상 대본(`data/수료생-인터뷰-대본`)의 대표 답변 — 세그먼트가 인스타 분석과 정확히 일치.

| 세그먼트 | 핵심 후기 (그대로 카피 소재) |
|---|---|
| **일반 입문** | "찍을 때 뭘 봐야 하는지 알게 됐어요. 전에는 그냥 셔터만 눌렀거든요." / "원장님이 제 사진 한 장 한 장 직접 설명해 주실 때가 제일 와닿았어요." |
| **창업·직업** | "외주 촬영 비용이 아예 없어졌어요. 제가 직접 다 찍으니까요." / "수료하고 바로 제품 촬영 의뢰가 들어왔어요." |
| **시니어 5060** | "퇴직하고 새로운 걸 배우고 싶었어요. 사진이 오래전부터 해보고 싶었던 거라." |

> ✅ **1:1 맞춤 · 즉시 피드백 · 실전 매출연결**이 세 세그먼트 공통 만족 포인트 → 리뉴얼·홍보 핵심 소구.

### 3-3. 시사점

1. **강점 = 1:1 맞춤 + 조명 실습 장비 + 실전 결과물(매출연결).** 리뷰·인터뷰 공통 → 홈페이지 핵심 소구점.
2. **국비지원**은 수강 결정의 결정타인데 노출이 약함 → 전면화 필요.
3. **후기 자산화 실패** — 만족한 수강생을 네이버·플랫폼 후기로 유도하는 장치가 없음. (인터뷰 영상은 얼굴 노출 부담 시 뒷모습·음성·AI 모델 대체 가능 → 실행 장벽 낮음)

## 5. 홍보 개선안

### 5-1. 문제 → 해결

| 문제 | 해결책 |
|---|---|
| 47K인데 홈페이지 유입/전환 미측정 | 프로필 링크 → 리뉴얼 홈(UTM 추적) |
| 반응 좋은 콘텐츠 유형이 산발적 | 3축(상업·스토리·AI) 주간 로테이션 |
| 후기가 인스타에만 갇힘 | 수료 시 네이버·플랫폼 후기 유도 이벤트 |
| 국비지원 노출 약함 | 릴스·홈·스토리 하이라이트 상시 노출 |
| 공지형 게시물 반응 낮음 | 스토리로 이전, 피드는 작품/스토리 중심 |

### 5-2. 전환 루프

```
인스타 릴스(작품·AI·스토리)
   → 프로필 링크(link in bio, UTM)
      → 리뉴얼 홈페이지
         → 국비지원+상담 폼 (전환 1%→5%)
            → 수강 → 수료 후기 유도
               → 후기가 다시 신규 유입 (선순환)
```

### 5-3. 즉시 실행 (7월 개강 캠페인)

- [ ] 프로필 링크를 리뉴얼 홈으로 교체 (UTM 파라미터 부착)
- [ ] 상단 고정글 = "7월 개강 + 국비지원 + 홈 링크"
- [ ] 스토리 하이라이트 정비: `개강안내` `국비지원` `수강후기` `작품` `오시는길`
