# 타임아카이브 — 원장님 운영 가이드

> 코드를 몰라도, 평소 쓰는 도구만으로 월 1~2회 수정이 가능합니다.

---

## 1. 작품 갤러리 수정 → 인스타그램만 올리면 끝

**원장님이 하실 일**
1. 학생·강사 작업물을 학원 인스타그램 계정에 올린다
2. 끝. 홈페이지 갤러리에 자동으로 나타난다

**최초 1회 설정 (개발자가 함)**
- 인스타그램 비즈니스 계정 연결
- Instagram Graph API 토큰 발급 → 환경변수 입력

> 인스타에서 게시물을 지우면 홈페이지에서도 자동으로 사라집니다.

---

## 2. 공지 · 개강 일정 수정 → 구글 시트에 한 줄 추가

**원장님이 하실 일**
1. 구글 시트를 연다
2. 한 줄 추가하거나 수정한다
3. 끝. 홈페이지 공지란이 자동으로 바뀐다

**시트 양식**

| 날짜 | 과정 | 내용 | 상태 |
|------|------|------|------|
| 2026-06-30 | 쇼핑몰 · 창업 · 취미반 | 신규 과정 개강 | 모집중 |
| 수시 | 1:1 개인 레슨 | 수시 접수 진행 중 | 상시 |
| 2026-08-01 | 영상 과정 | 영상반 신규 개강 | 모집중 |

**상태 뱃지 색상**
- `모집중` → 금색
- `상시` → 파란색
- `마감` → 회색

## 4. 자주 묻는 질문

**Q. 잘못 올리면 어떻게 되나요?**
인스타·시트에서 수정하면 홈페이지도 바로 따라 바뀝니다.

**Q. 전화번호·주소 같은 고정 정보는요?**
자주 안 바뀌므로, 바뀔 때 개발자에게 요청하시면 됩니다.

**Q. 컴퓨터를 못 다뤄도 되나요?**
인스타는 휴대폰으로, 시트는 한 칸 입력만 하면 됩니다.

## 5. 개발자 연동 체크리스트

```
인스타그램
- [ ] 인스타그램 비즈니스 계정 전환
- [ ] Graph API 액세스 토큰 발급
- [ ] INSTAGRAM_ACCESS_TOKEN 환경변수 입력
- [ ] INSTAGRAM_USER_ID 환경변수 입력

구글 시트
- [ ] 시트 생성 (위 양식대로)
- [ ] 파일 → 공유 → 웹에 게시 → CSV 선택
- [ ] NEXT_PUBLIC_NOTICE_SHEET_URL 환경변수 입력

Supabase
- [ ] consultations 테이블 생성
- [ ] RLS 정책 설정 (anon insert 허용)
- [ ] 환경변수 입력
- [ ] Sections3.jsx Consult 컴포넌트 주석 해제
```
