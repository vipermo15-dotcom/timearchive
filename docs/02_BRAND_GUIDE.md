# 타임아카이브 — Brand Guide

---

## 1. 브랜드 포지셔닝

**핵심 한 문장:**
> "사진 · 영상 · AI를 1:1로 배우는 실전 아카데미"

**메시지 계층:**
```
Level 1 (헤드라인)  →  "찍고, 편집하고, 팔리게 만든다"
Level 2 (서브)      →  사진 · 영상 · AI. 1:1 개인 레슨으로 처음부터 끝까지.
Level 3 (증거)      →  기초 없어도 됩니다. 실습 100%, 내 목적에 맞는 커리큘럼.
```

---

## 2. 두 시안 브랜드 방향

### 시안 A — 다크룸 (Darkroom)
- **컨셉:** 암실에서 사진이 현상되어 나타나는 순간
- **무드:** 어둡지만 따뜻함. 사진가의 작업 공간
- **카피 톤:** 강렬하고 직접적
- **헤드라인:** "사진은 시간을 멈추는 일"

### 시안 B — 아카이브 (The Archive)
- **컨셉:** 27년의 기록을 정리한 미술관 도록
- **무드:** 밝고 차분한 미술관. 격조와 신뢰
- **카피 톤:** 차분하고 격조 있는 에디토리얼
- **헤드라인:** "사진작가의 27년을 기록합니다"

## 4. 브랜드 보이스

### 쓰는 것 ✅
- "처음도 괜찮습니다. 첫날부터 직접 찍습니다."
- "27년 동안 현장에서 배운 걸 가르칩니다."
- "이론 없습니다. 첫날부터 카메라 듭니다."
- "배우는 사람의 목적이 다르면, 가르치는 방법도 달라야 합니다." (원장님 직접 인용)

### 쓰지 않는 것 ❌
- "타임은 입시학원이 아닙니다" (부정 메시지)
- "친절하게 설명해드립니다" (당연한 말)
- "다양한 교육 과정" (의미 없는 수식어)
- 느낌표 남용
