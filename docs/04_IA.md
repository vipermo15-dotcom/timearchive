## 1. 사이트맵

```
timearchive.co.kr/
├── /                        홈 (랜딩페이지)
├── /courses/
│   ├── /basic               기초 과정 (사진+영상+AI)
│   ├── /advanced-portrait   심화 — 인물·패션
│   ├── /advanced-product    심화 — 제품·커머스
│   └── /private             1:1 개인 레슨
├── /works/                  수강생 작품
│   ├── ?type=photo
│   ├── ?type=video
│   └── ?type=ai
├── /instructor/             강사(원장) 소개
├── /about/                  아카데미 소개
├── /blog/                   인사이트 블로그
└── /consultation/           1:1 상담 신청
```

```

    │
    ▼

    │
    ▼
[MosaicPortfolio] 12장 작품 그리드
    │
    ▼
[Instructor] 27년 경력, 1:1 직강

    │
```

## 3. 상담 신청 폼 설계

```
필드 (3개만 — 적을수록 전환율 ↑)
┌─────────────────────────────┐
│ 이름                         │
├─────────────────────────────┤
│ 연락처 (010-0000-0000)       │
├─────────────────────────────┤
│ 관심 과정  [드롭다운]         │
│ ○ 기초 과정 (사진+영상+AI)   │
│ ○ 심화 — 인물·패션          │
│ ○ 심화 — 제품·커머스        │
│ ○ 1:1 개인 레슨             │
│ ○ 아직 모르겠어요           │  ← 진입 장벽 낮추기
└─────────────────────────────┘
[무료 상담 신청하기]           ← Full Width, 56px

📞 바로 전화하기  02-545-5457  ← 5060 배려
```
