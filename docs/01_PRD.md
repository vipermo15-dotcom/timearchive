## 1. 비즈니스 맥락

### 시장 변화 (원장님 인터뷰 직접 반영)
- 사진업계 생성형 AI 도입 급속 확산
- 기존 사진학원은 AI 커리큘럼 전무 → **블루오션**
- 젊은 층(2030)은 비용 부담으로 이탈
- 2차 베이비부머 대거 은퇴 → 창업 수요 폭발
- 경쟁사(SAPPA 사파사진학원)는 입시·취창업 중심 — 차별화 가능

### 원장님 방향성 (통화 확인)
- 기존 사진 커리큘럼에 **영상 + AI 통합**
- 영상 파트 신규 론칭 (쇼츠·릴스·유튜브 수요 대응)
- **전 과정 1:1 개인 레슨** 유지
- **100% 실습** 원칙 고수
- **메인은 1:1 사진** — 커리큘럼은 두 번째
- 갤러리를 앞으로 빼서 인물·패션·제품이 돌아가게

## 2. 핵심 타겟 페르소나

### Persona A — 2차 베이비부머 창업자 ⭐ 주력
- **나이:** 52~65세
- **상황:** 작년~올해 퇴직, 새 수입원 탐색 중
- **니즈:** 쇼핑몰 창업, 유튜브 채널, 프리랜서 전환
- **강점:** 학습 의지↑, 교육비 여력↑, 시간 여유↑
- **불안:** "나이 들어서 배울 수 있을까?" "AI는 어렵지 않나?"
- **핵심 카피:** "카메라 한 번 잡은 적 없어도 됩니다. 3개월 후엔 팔리는 사진을 찍습니다."
- **CTA:** 1:1 무료 상담

### Persona B — 쇼핑몰·부업 창업자 (30~40대)
- **니즈:** 제품 사진 직접 찍어 비용 절감, 매출 향상
- **핵심 카피:** "사진 작가 안 써도 됩니다. 직접 찍으면 더 잘 팔립니다."

### Persona C — 영상 크리에이터 전환 (40~50대)
- **니즈:** 사진+영상+편집을 한 번에
- **핵심 카피:** "사진부터 영상, AI 편집까지. 한 곳에서 끝냅니다."

## 3. 핵심 차별점 (USP)

```
1. 사진 + 영상 + AI — 국내 사진학원 최초 통합 커리큘럼
2. 전 과정 1:1 개인 레슨 — 내 속도, 내 목적에 맞게
3. 100% 실습 — 첫날부터 카메라 들고 직접 찍는다
4. 27년 경력 원장 직강
```

## 4. 커리큘럼 구조 (원장님 인터뷰 직접 반영)

### 기초 과정
| 모듈 | 회차 | 내용 |
|---|---|---|
| 촬영 기초 | 5~6회 | 카메라 기본, 구도, 빛 읽기 |
| 영상 촬영 기초 | 4~5회 | 동영상 기초, 움직임, 앵글 |
| 프로그램 기초 | 4~5회 | Photoshop, Premiere, After Effects |
| AI 기초 | 추가 | 생성형 AI 활용 (차별화 포인트) |

### 심화 과정 (2트랙 선택)
- **트랙 A — 인물·패션:** 심화 촬영·조명·Premiere+AE 심화·AI 보정·합성 · 11회
- **트랙 B — 제품·커머스:** 제품 조명·스타일링·쇼핑몰 보정·AI 배경 · 11회

### 흐름
```
기초 (촬영+영상+프로그램+AI) → 심화 A 또는 B → 포트폴리오 완성
```
