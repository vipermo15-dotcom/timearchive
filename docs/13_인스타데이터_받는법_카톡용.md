# 원장님께 보낼 안내문 (인스타 인사이트 데이터 받기)

> 아래 "복사해서 카톡 전송" 블록을 그대로 원장님께 보내세요.
> 아이폰 기준 · 5060 눈높이 · 인스타 프로페셔널(비즈니스/크리에이터) 계정 전제.

---

## ✅ 방법 A — 스크린샷 (제일 쉬움, 바로 가능) · 추천

인사이트는 아이폰에서 파일로 내려받는 기능이 없어서 **화면 캡처**가 가장 빠르고 정확합니다.

**복사해서 카톡 전송 ↓**

```
원장님, 인스타 데이터 화면 캡처 부탁드려요 🙏
아래대로 3~4장만 찍어서 저한테 카톡으로 보내주시면 됩니다.

1) 인스타그램 앱 열고 → 오른쪽 아래 '내 프로필' 누르기
2) '프로페셔널 대시보드' (또는 '인사이트') 누르기
3) 기간을 '최근 30일'로 맞추기
4) 아래 화면들을 캡처해 주세요
   · 도달한 계정 수 화면
   · 참여(좋아요·저장·공유) 화면
   · 팔로워 화면 (연령·성별·지역 나오는 곳)
5) 잘 나온 게시물 2~3개는, 게시물 아래 '인사이트 보기' 눌러서 그 화면도 캡처

캡처 방법: 전원 버튼 + 볼륨 위 버튼 동시에 짧게 누르기
캡처한 사진을 카톡으로 저에게 보내주시면 됩니다 감사합니다!
```

---

## 📦 방법 B — 공식 전체 데이터 다운로드 (완전한 원본, 시간 걸림)

게시물 원본·전체 기록이 필요할 때. 이메일로 링크가 와서 **몇 시간~하루** 걸립니다.

**복사해서 카톡 전송 ↓**

```
원장님, 인스타 '내 정보 전체 다운로드' 방법이에요.
(파일이 이메일로 와서 시간이 조금 걸립니다)

1) 인스타 내 프로필 → 오른쪽 위 줄 세 개(≡) → '설정 및 개인정보'
2) '계정 센터' 누르기
3) '내 정보 및 권한' → '내 정보 다운로드'
4) '다운로드 요청' → 타임아카이브 계정 선택
5) 형식은 'HTML', 기간 '전체', 화질 '중간'으로 선택 후 요청
6) 등록된 이메일로 다운로드 링크가 옵니다 (몇 시간~하루)
7) 링크 눌러 압축파일(zip) 저장 → 그 파일을 저에게 카톡으로 보내주세요

* 이메일이 뭔지 모르시면, 그냥 방법 A(화면 캡처)만 보내주셔도 충분합니다!
```
