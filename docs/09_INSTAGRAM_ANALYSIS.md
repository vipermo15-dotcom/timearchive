- [ ] **② AI 사진 과정 섹션 신설/강조** — 최신 콘텐츠 반응이 좋고 차별화 포인트. 현재 커리큘럼(쇼핑몰/작가/취미/1:1)에 **AI 과정**을 별도 카드로
- [ ] **③ "입시학원 아님" 메시지 유지** — 강한 브랜드 보이스를 히어로 카피로. (예: "입시학원이 아닙니다. 현장 전문가의 학교입니다.")

- [ ] **⑦ 수강생 작품 갤러리** — "학생·선생님 촬영작" 소개글과 일관되게, 실제 작품 그리드 강조

### 5-2. 콘텐츠 필러 4종 (주간 로테이션)

| 요일 | 필러 | 포맷 | 목적 |

| 일 | **실전 팁**(조명/리터칭) | 릴스 | 저장 유도·전문성 |

